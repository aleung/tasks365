What is DateSlider ?
====================

It is a android widget which can be used exactly like the default DatePicker Dialog but present a neater interface


How To Install
==============

in the files you can find a sample application (i.e. activity) which
implements the DateSlider. Just copy the relevant files into your project
and you should be ready to go!

The relevant files are:

  In the src/ folder:
   - DateSlider.java
   - ScrollLayout.java
   - TimeView.java

  In the res/ folder:
   - drawable/slider_nosnap.png
   - layout/dateslider.xml
   - layout/dialogtitle.xml
   - layout/scroller.xml
   - values/strings.xml

moreover you can use any of the DateSlider implementations you will find in the src/ folder.
